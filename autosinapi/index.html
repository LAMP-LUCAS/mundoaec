Comece a Usar em 3 Passos

Siga os passos abaixo e faça sua primeira consulta à API em menos de 5 minutos.

1. Crie sua Conta

Cadastre-se no plano Free para começar.

2. Gere sua Chave

Acesse seu painel e copie sua chave de API.

3. Faça a Chamada

Use a chave no header de sua requisição.

Exemplos de Requisição

cURL (bash)

curl -X GET "https://api.mundoaec.com/autosinapi/v1/insumos?q=cimento" \
-H "Authorization: Bearer SUA_CHAVE_DE_API"

Python (requests)

import requests

headers = {
    'Authorization': 'Bearer SUA_CHAVE_DE_API'
}
params = {
    'q': 'cimento'
}
response = requests.get('https://api.mundoaec.com/autosinapi/v1/insumos', headers=headers, params=params)
print(response.json())

JavaScript (fetch)

fetch('https://api.mundoaec.com/autosinapi/v1/insumos?q=cimento', {
    headers: {
        'Authorization': 'Bearer SUA_CHAVE_DE_API'
    }
})
.then(response => response.json())
.then(data => console.log(data));

Perguntas Frequentes

Com que frequência os dados do SINAPI são atualizados na API?

Nossa plataforma verifica e atualiza a base de dados mensalmente, no mesmo dia em que a Caixa Econômica Federal disponibiliza os novos arquivos, garantindo que você sempre tenha os dados mais recentes.

O que acontece se eu exceder o limite do meu plano?

Para o plano Free, requisições adicionais serão enfileiradas com menor prioridade. Para os planos pagos, você será notificado e terá a opção de fazer um upgrade ou aguardar o próximo ciclo de faturamento.

Qual o SLA de disponibilidade da API?

O SLA varia conforme o plano. O plano Pro tem 99% de uptime garantido, enquanto o Business oferece 99.5%. O plano Enterprise pode ter um SLA customizado em contrato. O plano Free não possui garantia de SLA.

Posso cancelar minha assinatura a qualquer momento?

Sim. Você pode cancelar sua assinatura a qualquer momento através do seu painel de controle. O acesso permanecerá ativo até o final do ciclo de faturamento já pago.
